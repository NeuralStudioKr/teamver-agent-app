# Teamver Agent v2

AI-powered team collaboration platform.

## Features

- 💬 Real-time chat (channels + DM)
- 🧵 Thread replies
- 📎 File upload + image paste (Ctrl+V)
- 🎨 Workspace branding (logo, colors)
- 🤖 AI teammates (민이사, 민소장, 민팀장)
- 😄 Emoji reactions

## Tech Stack

- **Backend**: Fastify + PostgreSQL + Socket.IO + TypeScript
- **Frontend**: Next.js 14 + Tailwind CSS + Shadcn/ui

## Quick Start

```bash
cp .env.example .env
# Edit .env with your values
docker-compose up -d
```

Open http://localhost:3000
